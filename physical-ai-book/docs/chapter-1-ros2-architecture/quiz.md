# Chapter 1 Quiz: ROS 2 Architecture & Core Concepts

Test your understanding of ROS 2 architecture and the concepts of embodied AI.

## Multiple Choice Questions

### Question 1
What is the primary difference between traditional AI systems and embodied AI systems using ROS 2?

A) Traditional AI systems are more complex than embodied AI systems
B) Traditional AI systems operate in virtual environments while embodied AI systems interact with the physical world
C) Traditional AI systems are faster than embodied AI systems
D) Traditional AI systems use different programming languages

### Question 2
Which of the following best describes the "robotic nervous system" concept in ROS 2?

A) A hardware component inside the robot
B) A specific software library for neural networks
C) The metaphorical description of how ROS 2 connects AI agents to physical robot control
D) A type of sensor used in robotics

### Question 3
What does DDS stand for in the context of ROS 2?

A) Distributed Data System
B) Data Distribution Service
C) Dynamic Development Suite
D) Digital Design Software

### Question 4
Which communication pattern does ROS 2 use for asynchronous communication between nodes?

A) Client-Server
B) Request-Response
C) Publish-Subscribe
D) Peer-to-Peer

### Question 5
What is the main purpose of Quality of Service (QoS) settings in ROS 2?

A) To make the code run faster
B) To control how messages are delivered, including reliability and durability
C) To reduce memory usage
D) To increase the number of supported nodes

## Answers

1. B) Traditional AI systems operate in virtual environments while embodied AI systems interact with the physical world
2. C) The metaphorical description of how ROS 2 connects AI agents to physical robot control
3. B) Data Distribution Service
4. C) Publish-Subscribe
5. B) To control how messages are delivered, including reliability and durability
